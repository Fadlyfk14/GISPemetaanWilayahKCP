Sistem Informasi Geografis (SIG) berbasis web yang dirancang untuk memetakan, mengelola, dan menganalisis sebaran titik lokasi **Kantor Cabang Pembantu (KCP)** dan **Agen Pos** di bawah naungan **Kantor Cabang (KC) Kotamobagu 95700**.

---

## ğŸ“ Deskripsi Proyek
Website ini dikembangkan sebagai alat bantu strategis dalam perencanaan cakupan layanan pos. Dengan adanya sistem ini, Kantor Cabang dapat memastikan distribusi layanan menjangkau masyarakat secara optimal dan merata. 

Bagi masyarakat, platform ini berfungsi sebagai portal informasi publik untuk mempermudah pencarian outlet pos terdekat berdasarkan data lokasi yang akurat dan terukur.

## ğŸš€ Fitur & Tahapan Pengembangan

### 1. Pengumpulan & Persiapan Data Spasial
* Validasi koordinat titik lokasi untuk akurasi posisi peta digital.
* Penyusunan basis data yang mengintegrasikan data statis (alamat) dan data dinamis (status operasional).

### 2. Implementasi Peta Digital
* Peta interaktif dengan fitur navigasi *zoom* dan *drag*.
* Optimasi performa untuk memuat banyak titik lokasi secara responsif.

### 3. Analisis Spasial & Clustering
* Logika pemfilteran berdasarkan status operasional kantor.
* **Marker Clustering**: Pengelompokan otomatis titik lokasi untuk menghindari penumpukan visual pada peta.

### 4. Integrasi Data Eksternal
* Sinkronisasi dengan basis data Manajemen SDM.
* Menampilkan informasi petugas, kontak darurat, dan data pimpinan cabang secara *real-time*.

### 5. Visualisasi & Dashboard SIG
* **Color Coding**: Indikator warna untuk memantau status operasional secara cepat.
* **Dashboard Monitoring**: Ringkasan kondisi infrastruktur (Listrik & Jaringan Internet) di setiap titik.
